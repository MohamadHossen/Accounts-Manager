# 📁 Accounts Manager System

برنامج بسيط لإدارة الحسابات اليومية مكتوب بلغة C++

## ✨ الوظائف المتوفرة:
- ➕ إضافة حساب جديد
- ❌ حذف حساب
- ✏️ تعديل حساب
- 🔍 بحث عن حساب
- 📋 عرض كل الحسابات

## 📌 أنواع الحسابات المدعومة:
- Gmail
- Facebook
- WhatsApp
- Network

## 🎯 طريقة الاستخدام:
1. شغل البرنامج
2. اختار رقم العملية اللي تبيها
3. أدخل البيانات المطلوبة
4. البرنامج يحفظ لك كل شي في ملف Accounts.txt
